# 青青生日快乐

一个静态生日惊喜网站：信封开场、文字谜解密、唱片播放器、电子生日信和手写信展示。

## 本地预览

```powershell
python -m http.server 4173
```

然后打开：

```text
http://localhost:4173/
```

## 手写信替换

写好手写信后，拍一张清晰、正面、光线均匀的照片，保存为：

```text
letter.jpg
```

刷新网页后，最终页面会自动显示它。

## 手写信建议

适合控制在 120-180 字，重点写普通朋友之间真诚、轻松的祝福：

- 祝她生日快乐。
- 谢谢她愿意和你聊天。
- 说这个小网站只是想让她今天开心一点。
- 祝她新的一岁平安、开心、顺利、少一点烦恼。

建议避免太暧昧或压力很大的表达，例如“特别喜欢你”“一直陪着你”“只属于你”“你对我来说最特别”。可以写得朴素一点，反而更真诚。

## 音乐与谜题

- 背景音乐使用 `assets/HappyBirthday.mp3`。
- 右上角唱片按钮可以播放/静音。
- 谜题答案是 `0526`，但需要先用四个文字线索把“圆月、午风、耳语、柳枝”读成四位口令。
